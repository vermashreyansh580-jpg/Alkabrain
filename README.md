# AlkaBrain — 3D Product Showcase

Immersive 3D landing page for AlkaBrain, an AI brain assistant. Built with React, Vite, React Three Fiber, and Framer Motion.

The "Launch AlkaBrain" CTA opens the live app at https://shrey77777-alkabrain.hf.space/.

## Stack

- React 19 + Vite
- React Three Fiber + drei + postprocessing (interactive 3D hero)
- Framer Motion (scroll animations)
- Tailwind CSS v4
- pnpm workspace monorepo

## Run locally

```bash
pnpm install
pnpm --filter @workspace/alkabrain run dev
```

The 3D web app lives in `artifacts/alkabrain/`.
